# 3D Mobile Chess — Unreal (WIP)

## Quick goal
Polished 3D chess for mobile with Stockfish AI, animated moves, VFX and sounds.

## How to run (developer)
1. Install Unreal Engine 5.x (matching engine version).
2. Clone repo and `git lfs install`.
3. Open `<Project>.uproject` in Unreal Editor.
4. Generate Visual Studio / Xcode project and build.

## Branching / workflow
- `main` — release-ready
- `develop` — integration
- `feature/*` — feature branches, open PRs to `develop`
